# SanaVita
SanaVita is a software platform developed as a graduation project. The system is designed to collect, process, analyze, and store medical data in order to improve diagnosis, treatment, and overall quality of patient care.
It integrates web technologies, IoT devices, and electronic medical records into a single ecosystem, enabling efficient remote interaction between patients and healthcare professionals.
